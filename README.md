# Logical-Parkour
This is a game where you have to solve Puzzles and Complete Parkour.

WASD to move and space to jump. (you can also use arrows)
You have to go in the portal and wait around a half a second to teleport you.
The sensitivity is set to high so i recomend lowering it down
Purple-Pink rectangle makes you jump. It is a trampoline in the game
Lava Kills you
you have to find pressure Plate to activate the portal.
I used AI to help me debug if there was a compiler error. 
The thing that says LOGICAL PARKOUR on the main menu is made by me and I used an app where I type, and they make me a cool title with different cool fonts.
